Laboratory Exercises (Inter-Midterm Period)

Below are the laboratory activities 1 through 5, including explanations of achievements and outcomes. Screenshots and demos are linked via Google Drive for detailed viewing.

Laboratory Activity 1

Card Hover Effect to Show Paragraph (CSS Only)

Implemented a hover effect on a card or container using pure CSS transitions. When the user hovers over the card, a hidden paragraph fades in and slides up smoothly without any JavaScript, demonstrating efficient CSS animations for enhanced user interaction.

Laboratory Activity 2

Button Toggle to Show/Hide Paragraph (JavaScript)

Created a button that toggles the visibility of a paragraph using JavaScript event listeners. On the first click, the paragraph appears; on subsequent clicks, it disappears, showcasing basic DOM manipulation and state management for interactive content.

Laboratory Activity 3

Button Toggle for Dark/Light Mode (JavaScript)

Developed a button that switches the page between dark and light modes using JavaScript to dynamically update CSS classes and styles. Clicking the button toggles the theme back and forth, improving accessibility and user preference customization.

Laboratory Activity 4

Button Toggle for Dark Mode and Paragraph Change (JavaScript)

Built a button that, on click, switches to dark mode and replaces the original paragraph with new content using JavaScript. Clicking again reverts to light mode and restores the first paragraph, combining theme switching with dynamic text updates for a more engaging interface.

Laboratory Activity 5

Dynamic Table List with Add, Highlight, and Remove Functions (JavaScript)

Designed an input table where users enter text and click an "Add" button to append items to a dynamic list using JavaScript. The list supports highlighting selected items, individual removal, "Remove All" to clear the list, and "Highlight All" to emphasize everything, illustrating advanced DOM manipulation and event handling.
